# Simple url crawler

## Installation

* install packages with npm or yarn:
``yarn``

* build the app:
``npm run build``

* to build interactively use
``npm run build:watch``

## Development

* Run tslint:
``npm run lint``

* Run the watch build and test interactively:
``npm run dev``

## Run
* ``node ./dist/app.js https://your-url-goers-here/``

* to test this program on 'https://wiprodigital.com':
``npm run exec``
